### Описание
Анализатор страниц - это веб-сервис, который делает запросы к указанным сайтам, анализирует их и сохраняет полученные данные в созданной базе данных. Приложение создано на основе фреймворка на Flask.

### Загрузка и установка:
1. Загрузите приложение с GitHub, используя 
[link] (https://github.com/OlegGolubev74/python-project-83)
2. Создайте файл '.env' и введите в него данные для подключения к базы данных.
3. Введите 'make biuld', чтобы подключить базу данных к проекту.

### Запуск приложения локально в режиме разработки
* Введите 'make dev'

### Запуск приложения на Render.com
* Введите 'render-start'
